## Overlay Props

| 名称                 | 类型      | 说明     | 默认值  |
| -------------------- | --------- | -------- | ------- |
| modelValue / v-model | `boolean` | 是否打开 | `false` |
| fullscreen           | `boolean` | 全屏显示 | `false` |

## Overlay Events

| 名称  | 说明         | 类型         |
| ----- | ------------ | ------------ |
| click | 点击遮罩触发 | `() => void` |
