## Resize Props

| 名称       | 类型                                     | 说明                | 默认值 |
| ---------- | ---------------------------------------- | ------------------- | ------ |
| directions | `'top' \| 'right' \| 'bottom' \| 'left'` | 可拖拽边            | -      |
| size       | `number`                                 | 拖拽边大小, 单位 px | `8`    |

## v-resize

| 参数      | 说明       | 类型                                     |
| --------- | ---------- | ---------------------------------------- |
| modifiers | 可拖拽的边 | `'top' \| 'right' \| 'bottom' \| 'left'` |
